## Boot.dev HTTP Protocol in Go

## Aim
In this project i created an HTTP 1.1 compliant server without using http package with features like parsing request lines, headers, body etc from text (byte data), Chunked Encoding with Trailer support, Custom handlers

## Why
As an Electrical and Electronical Engineer working in the IT industry, i want to not just use the newer tools and technologies but over time learn and understand the inner workings of the tools and technologies i use in my day to day work to be able to better understand the problems i face and find better solutions.
